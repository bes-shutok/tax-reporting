# Crypto Origin Archive (Chain And Operator Source Mapping)

Local archive of chain/operator origin material used to resolve crypto `País da Fonte`, operator country, and chain metadata.

## Terminology

- Official extract: a local Markdown capture of relevant facts from an official web page when no PDF origin is available.
- Repository override: a documented local mapping choice kept visible because it is narrower than the upstream platform's global footprint.

## Contents

- `official/`: extracted official source material for chains and operators.
- `public/`: supporting public-record extracts used when an official site does not state domicile directly.
- `sources.md`: manifest of the archived sources in this folder.
- `operator_chain_origin_registry.md`: current mapping registry used by the repository.
- `mapping_decision_log.md`: reasoning log explaining why each mapping was chosen and whether it is official, inferred, or a repository override.

## Scope Boundary

- This folder is for chain/operator origin evidence only.
- Portugal-specific tax-law sources, AT forms, and filing rules belong in `../portugal-crypto-tax/`.
